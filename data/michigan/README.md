## Status: Requires browser/manual download

The Michigan LARA system uses an Accela-based report generator that requires
a browser form submission to download the data.

**Report URL:** https://aca-prod.accela.com/MILARA/Report/ReportParameter.aspx?module=Licenses&reportID=32411&reportType=LINK_REPORT_LIST

**Steps needed:**
1. Visit the URL above in a browser
2. Select report parameters (Real Estate Brokers/Salesperson)
3. Click download (Excel/PDF output)
4. Convert to CSV and normalize

**Alternative:** Submit a FOIA request to LARA for bulk licensee data.

No open data portal dataset found on data.michigan.gov.
